vim支持在K端剪N板共享

2012-06-03 16:46:15   查看原文>>

碜http://basiccoder.com/vim-copy-paste-related.html

查看Vim是否支持clipboard和xterm_clipboard
[root@fedora fedora]# vim --version|grep clipboard
-clientserver -clipboard +cmdline_compl +cmdline_hist +cmdline_info +comments 
 -xterm_clipboard -xterm_save 

#可以看出不支持@功能

[root@fedora fedora]# gvim --version|grep clipboard
+clientserver +clipboard +cmdline_compl +cmdline_hist +cmdline_info +comments 
+xsmp_interact +xterm_clipboard -xterm_save 

#gvim是支持的
到http://www.vim.org/download.php下d源a重新g,我下d的是vim-7.3.tar.bz2
./configure --prefix=/usr \
--sysconfdir=/etc \
--enable-tclinterp \
--enable-pythoninterp \
--enable-perlinterp \
--enable-rubyinterp \
--enable-cscope \
--enable-multibyte \
--enable-xim \
--enable-gtk2-check \
--enable-fontset \
--with-x

[root@fedora vim73]# make

[root@fedora vim73]# make install
好了,@釉诮K端就可以使用"+yM行}u了,跟gvim一
