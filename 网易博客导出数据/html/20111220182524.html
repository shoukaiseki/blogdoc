cd /home/fedora/weblogic
unzip wloc103_linux32.zip -d /tmp/database
查看java版本
java -version
我的1.6.0_22版本的
cd /tmp/database


1.1 部署前准备
a、创建weblogic用户组。
        groupadd weblogic
        useradd -g weblogic weblogic (增加ぬ,密a)
        passwd weblogic（修改weblogic密码,不用更改密a就可以不用绦写司洌
b、创建/oracle/weblogic目录,修改目录所有者权限
mkdir /oracle
        mkdir /oracle/weblogic
        chown CRf weblogic:weblogic /oracle/weblogic  x予weblogic用粼L/oracle/weblogic的L嘞

1.2 安装WebLogic10
赋予.bin可执行权限： chmod a+x wloc103_linux32_jdk6.bin
su weblogic
执行./wloc103_linux32_jdk6.bin
      进入控制台安装的欢迎界面，键入[Next]继续
      此时出现授权许可协议，读完后键入[Yes]继续
      提示选择一个BEA主目录:/oracle/weblogic
             输入 ./server103_linux32.bin
然后根据提示一一选择
即可没有什么特别的地方

1.3新建domain
使用WebLogic用户登录系统：
###因之前用root安b的weblogic,所以/weblogic下有root嘞薏拍茉L的文件A
###cd /oracle 
###chown -Rf weblogic:weblogic weblogic/  修退儆艚M
[root@linux-test /]# su  weblogic
[weblogic@linux-test ~]$ cd /oracle/weblogic/wloc_10.3/common/bin/

[weblogic@linux-test bin]$ pwd
/oracle/weblogic/wlserver_10.3/common/bin
[weblogic@linux-test bin] ./config.sh
http://localhost:7001/console
