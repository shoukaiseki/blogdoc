2017-05-27 12:48:44   查看原文>>
安装 nautilus 目录管理器
sudo apt-get install nautilus

问题处理方案：
先把 eclipse 关了，打开终端
ALT +F2 运行 mate-terminal
再在 终端运行eclipse
eclipse运行后点击项目右键 选择 open in explorer
